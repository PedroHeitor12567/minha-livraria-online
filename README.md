# 📚 Minha Livraria Online

Um projeto de e-commerce para venda de livros online, com foco em simplicidade, usabilidade e boa experiência do usuário.

---

## 🚀 Sobre o Projeto

A **Minha Livraria Online** é uma aplicação web que permite:

* 📖 Visualizar livros disponíveis
* 🔎 Buscar por título, autor ou categoria
* 🛒 Adicionar itens ao carrinho
* 💳 Simular compras
* 👤 Gerenciar usuários (login/cadastro)

## ▶️ Como Executar o Projeto

### Pré-requisitos

* Node.js instalado
* Gerenciador de pacotes (npm ou yarn)

### Passos

```bash
# Clone o repositório
git clone https://github.com/seu-usuario/minha-livraria-online.git

# Acesse a pasta
cd minha-livraria-online

# Instale as dependências
npm install

# Inicie o projeto
npm run dev
```

---

## 🔧 Configurações

Crie um arquivo `.env` na raiz com:

```env
PORT=3000
DATABASE_URL=sua_string_de_conexao
JWT_SECRET=sua_chave_secreta
```
